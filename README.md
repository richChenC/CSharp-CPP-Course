# C# 与 C++ 完整课程

这是一个面向初学者到进阶学习者的课程项目，包含两套完整课程：

- `C#课程`
- `C++课程`

每一课都尽量保持统一结构：

- `README.md`：本课学习目标与知识点
- `Notes.md`：课堂笔记与重点整理
- `Practice.md`：练习题
- `Examples/`：示例代码与代码讲解

## 课程特点

- 全程中文讲解
- 示例代码带详细中文注释
- 覆盖基础语法、面向对象、调试思维、文件与异常、现代语言特性
- 包含综合项目，适合练手与整理作品集

## 本地开发环境

项目已经针对 VS Code 做过调试配置，建议使用：

- `ms-vscode.cpptools`
- `ms-dotnettools.csdevkit`
- `ms-dotnettools.csharp`

不建议使用会自动改写调试配置的第三方 C/C++ 运行扩展。

## 目录说明

### C++课程

从基础语法一路到现代 C++、智能指针和调试实战，并包含通讯录管理系统综合项目。

### C#课程

从基础语法、集合、面向对象一路到委托、事件、异步与调试实战，并包含学生成绩管理系统综合项目。

## 建议学习方式

1. 先阅读每课 `README.md`
2. 再看 `Notes.md`
3. 自己动手运行 `Examples/` 中代码
4. 完成 `Practice.md`
5. 尝试在原代码基础上改功能，再做断点调试

## 仓库用途

这个仓库既可以作为课程资料仓库，也可以作为个人学习记录、面试项目整理仓库的基础版本。
