# 学习记忆总结

更新时间：2026-05-14

我已经学习到 C++ 课程第 13 课：现代 C++ 与 auto、lambda。

当前重点不是把所有语法一次背完，而是把 C 语言基础迁移到 C++ 的思维方式：

- C 语言更偏向函数和数据分开处理。

## 已经建立的核心记忆

### 1. cout 与 printf 的区别

`cout <<` 是 C++ 的输出流写法。

```cpp
cout << "hello" << endl;
```

可以理解成：把内容依次送到控制台。

`printf` 是 C 语言风格，需要格式控制符。

```c
printf("%d", age);
```

现在学习 C++ 时，优先使用 `cout` 和 `cin`。

### 2. class、对象和封装

`class` 是把数据和函数放在一起。

```cpp
class Dog {
public:
    void speak() {
        cout << "狗狗叫" << endl;
    }
};
```

对象就是类创建出来的具体东西。

```cpp
Dog d;
d.speak();
```

封装的核心是：把对象自己的数据和行为放在一起，外部通过公开接口使用。

### 3. 继承和多态

继承表示子类拥有父类的能力。

```cpp
class Dog : public Animal {
};
```

多态的关键写法是：

```cpp
Animal* a = new Dog();
a->speak();
```

如果 `speak()` 是 `virtual` 虚函数，那么虽然指针类型是 `Animal*`，实际执行的是 `Dog::speak()`。

一句话记忆：

> 虚函数让父类指针调用函数时，能自动找到子类真正的版本。

### 4. new、delete 和智能指针

`new` 会申请内存，`delete` 负责释放内存。

```cpp
Animal* a = new Dog();
delete a;
```

如果忘记 `delete`，内存会一直被占用，叫内存泄漏。

现代 C++ 更推荐智能指针：

```cpp
unique_ptr<Animal> a = make_unique<Dog>();
a->speak();
```

智能指针会自动释放内存，减少忘记 `delete` 的风险。

### 5. vector、sort 和 STL

`vector` 可以先理解成 C 语言数组的升级版。

```cpp
vector<int> nums = {5, 2, 9, 1};
```

`sort` 是标准库算法。

```cpp
sort(nums.begin(), nums.end());
```

核心记忆：

- 容器负责存数据。
- 算法负责处理数据。
- `begin()` 和 `end()` 告诉算法处理哪一段。

### 6. 文件读写

写文件：

```cpp
ofstream out("note.txt");
out << "C++ 文件写入示例" << endl;
```

读文件：

```cpp
ifstream in("note.txt");
string line;
getline(in, line);
```

`cout` 是输出到控制台，`ofstream` 是输出到文件。

### 7. auto 和 lambda

`auto` 是让编译器自动推断类型。

```cpp
for (const auto& value : numbers) {
    cout << value << " ";
}
```

`const auto&` 的意思是：只读取，不修改，不复制。

`lambda` 是临时写在原地的小函数。

```cpp
sort(numbers.begin(), numbers.end(), [](int left, int right) {
    return left > right;
});
```

这段表示：按从大到小排序。

一句话记忆：

> auto 少写复杂类型，lambda 直接写临时规则。

## 当前调试能力记忆

### F5 调试

使用官方 C++ 调试配置：

```text
COURSE_ONLY_DEBUG_CURRENT_CPP
```

调试时重点看：

- 黄箭头停在哪一行。
- 左侧变量区的值。
- 调用栈显示当前执行到哪个函数。
- F10 和 F11 的区别。

### F10 和 F11

`F10`：逐过程，执行当前这一行，但不进入函数内部。

`F11`：单步，如果当前行调用了函数，就进入函数内部。

记忆：

```text
F10 = 看流程
F11 = 进函数
```

### 调试第 9 课时要看什么

重点代码：

```cpp
Animal* a = new Dog();
a->speak();
delete a;
```

观察点：

- 执行 `new Dog()` 后，变量 `a` 是否有地址。
- 执行 `a->speak()` 时，是否进入 `Dog::speak()`。
- 执行 `delete a` 后，程序是否正常结束。
