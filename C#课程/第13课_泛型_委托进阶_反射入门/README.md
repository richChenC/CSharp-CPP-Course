# 第13课_泛型_委托进阶_反射入门

## 本节目标
理解 C# 为什么需要泛型，进一步理解委托的本质，并初步认识反射。

## 本节知识点
- 泛型方法
- `Func<>` 委托
- 反射基础
- `typeof`
- `GetMethods`

## 重点理解
- 泛型是“写一套逻辑，适配多种类型”
- 委托是“把方法当成数据传来传去”
- 反射是“程序运行时查看类型信息”

## 调试训练重点
- 观察泛型方法在不同类型参数下的行为
- 看委托变量里到底装了哪个方法
- 断点查看 `methods.Length`

## 面试高频点
- 泛型的好处
- `Func<>` 和普通方法的关系
- 反射有哪些典型场景

## 本节小结
这节课是从“会写代码”走向“理解框架底层思想”的第一步。
