# 逐行代码讲解

| 3 | `// 引入泛型集合命名空间，List&lt;T&gt; 依赖这里。` | 注释：引入泛型集合命名空间，List<T> 依赖这里。 |

| 18 | `}` | 代码块结束，表示当前作用域到这里结束。 |
