# C# 课程介绍

这是一套从基础到进阶、从语法到调试、从练习到项目的 `C#` 系统课程。它不是只讲“怎么写”，还会带你理解“为什么这样写”“错了怎么查”“项目里怎么组织代码”。

## 这套课教什么

课程内容覆盖：

- C# 基础语法
- 变量、类型、输入输出、流程控制
- 方法、代码复用、面向对象
- 封装、继承、多态
- 数组、列表、字符串、集合
- 泛型、LINQ、委托、事件
- 异常处理、文件操作、异步编程
- 反射入门、调试实战
- 综合项目开发

## 这套课怎么教

整套课按下面的学习逻辑来设计：

1. 先讲清知识点的用途
2. 再给出能直接运行的最小示例
3. 每行代码都写中文注释
4. 再用 `CodeWalkthrough.md` 逐行拆解
5. 给练习、给作业、给复盘模板
6. 在关键章节安排专门的调试训练

## 你会学到什么能力

学完这套课，你应该能逐步做到：

- 看懂常见 C# 基础代码
- 自己写方法、类、集合处理逻辑
- 理解委托、事件、异步这些常见能力
- 会看异常信息，会用断点查错
- 能完成一个结构清晰的控制台项目

## 适合谁学

这套课适合：

- 零基础或基础薄弱的学习者
- 想学 Unity / .NET / Windows 开发的人
- 想以后做后台系统、业务系统的人
- 想把 C# 基础系统化的人

## 建议学习顺序

每节课建议按这个顺序推进：

1. 阅读本节 `README.md`
2. 打开并运行 `Examples/Program.cs`
3. 结合 `CodeWalkthrough.md` 逐行理解
4. 完成 `Practice.md`
5. 把错题、报错、理解记录进 `Notes.md`

## 教学重点

这套课会特别强调：

- 理解对象和数据怎么流动
- 遇到异常时先定位再修改
- 不只会“写出来”，还要会“讲清楚”
