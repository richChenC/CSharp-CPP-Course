# 逐行代码讲解

这个文件用于配合示例代码学习。代码文件里已经有中文注释，这里再按行号解释每一行，方便检查和复习。

| 行号 | 代码 | 说明 |
|---:|---|---|
| 1 | `// 引入基础命名空间，后面要使用 Console 输出内容。` | 注释：引入基础命名空间，后面要使用 Console 输出内容。 |
| 2 | `using System;` | 引入命名空间或声明使用范围，减少后续代码的书写成本。 |
| 4 | `// 调用 Add 方法，并把返回结果输出到控制台。` | 注释：调用 Add 方法，并把返回结果输出到控制台。 |
| 5 | `Console.WriteLine(Add(3, 5));` | 向控制台输出提示或结果，是用户看到程序反馈的地方。 |
| 7 | `// 定义一个静态本地方法，接收两个整数并返回它们的和。` | 注释：定义一个静态本地方法，接收两个整数并返回它们的和。 |
| 8 | `static int Add(int a, int b)` | 声明变量、函数或返回类型，是程序保存数据和组织逻辑的关键。 |
| 9 | `{` | 代码块开始，表示接下来的代码属于同一个作用域。 |
| 10 | `    // 把两个参数相加，并把结果交还给调用者。` | 注释：把两个参数相加，并把结果交还给调用者。 |
| 11 | `    return a + b;` | 返回结果或结束当前函数。 |
| 12 | `}` | 代码块结束，表示当前作用域到这里结束。 |
