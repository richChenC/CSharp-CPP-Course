# 第14课_智能指针与资源管理

## 本节目标
理解裸指针为什么危险，学会使用 `unique_ptr` 管理资源，并建立 RAII 思维。

## 本节知识点
- 裸指针的风险
- `unique_ptr`
- `make_unique`
- 自动释放资源
- RAII 基本思想

## 重点理解
- “谁申请资源，谁负责释放”在真实项目里很容易写漏
- 智能指针的重点不是语法，而是生命周期管理
- 调试时要重点观察对象何时创建、何时销毁

## 调试训练重点
- 在构造函数和析构函数打断点
- 观察对象离开作用域后是否自动析构
- 比较裸指针思路和智能指针思路的差别

## 面试高频点
- `unique_ptr` 和 `shared_ptr` 的区别
- 为什么推荐 `make_unique`
- RAII 是什么
- 智能指针能完全替代裸指针吗

## 本节小结
智能指针是在帮你把“资源释放”这件容易忘的事，交给语言机制自动完成。
