# 逐行代码讲解

这节课不是看“对的代码”，而是学“错的代码怎么查”。

| 行号 | 代码 | 说明 |
|---:|---|---|
| 15 | `vector<int> scores{95, 88, 76};` | 数组实际只有 3 个元素。 |
| 18 | `int index = 5;` | 这里故意给了错误下标。 |
| 24 | `int value = scores.at(index);` | `at` 会做边界检查，越界时抛异常。 |
| 28 | `catch (const exception& ex)` | 捕获异常，避免程序直接崩掉。 |

## 建议你这样调试
1. 在 `index = 5;` 处打断点，看 index 值。
2. 在 `scores.at(index)` 处打断点，看 scores 长度。
3. 想一想：合法下标范围是多少。
4. 把 `index` 改成 `2`，再运行一次验证修复结果。
