# C++ 课程介绍

这是一套面向初学者到进阶学习者的 `C++` 系统课程。课程不是只带你记语法，而是按“概念理解 -> 代码阅读 -> 自己改写 -> 调试排错 -> 小项目落地”这条线来教。

## 这套课教什么

课程内容覆盖：

- C++ 基础语法
- 输入输出、变量、类型、运算符
- 分支、循环、函数、引用、作用域
- 指针、内存、结构体、类
- 封装、继承、多态
- STL 容器与算法
- 文件操作、异常处理、现代 C++
- 智能指针、资源管理、调试实战
- 综合项目开发

## 这套课怎么教

教学方式不是“上来甩定义”，而是按下面这个顺序：

1. 先用最小示例把知识点跑起来
2. 每一行代码都配详细中文注释
3. 再配 `CodeWalkthrough.md` 做逐行讲解
4. 用 `Practice.md` 让你自己动手改和写
5. 用 `Notes.md` 逼你复盘和总结
6. 在关键课程里加入调试训练，培养你独立排错能力

## 你会学到什么能力

学完这套课，你应该逐步具备这些能力：

- 看懂基础到中级 C++ 代码
- 能自己写出函数、类、容器操作代码
- 知道常见 bug 为什么出现
- 会用断点、单步执行、变量观察来调试
- 能完成一个中小型控制台项目

## 适合谁学

这套课适合：

- 刚开始学编程的人
- 想转向算法、竞赛、系统开发的人
- 想做游戏、图形、嵌入式方向的人
- 想把 C++ 基础打扎实的人

## 建议学习顺序

每节课建议按这个顺序学：

1. 先看本节 `README.md`
2. 再打开 `Examples/main.cpp`
3. 运行程序，观察输出
4. 对照 `CodeWalkthrough.md` 逐行理解
5. 完成 `Practice.md`
6. 把自己的理解写进 `Notes.md`

## 教学重点

这套课特别强调三件事：

- 不死记硬背语法
- 不跳过报错和 bug
- 不只会抄代码，要能解释为什么这么写
